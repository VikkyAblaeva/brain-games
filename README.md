## Краткое описание: :zap:

В данном мини-проекте представлено **5 мини-игр с пользователем**. В каждой игре выполнено приветствие, обращение к пользователю по имени. Количество раундов во всех играх равно трём.

```
   - brain-even: спрашивает у пользователя, является ли число чётным;
   - brain-calc: предлагает пользователю выполнить операцию над двумя числами (+,-,*);
   - brain-gcd: предлагает пользователю определить наибольший общий делитель двух чисел;
   - brain-progression: запрашивает недостающий элемент арифметической прогрессии;
   - brain-prime: спрашивает у пользователя, является ли число простым.
```
